Rome

  • Purpose:
    1. To use ANTLR and the visitor pattern to dynamically parse text input via SMS

  • System Model Explanation:
    1. The user sends an SMS text message to the Twilio number
    2. The message is then captured by the Flask web app via webhook
    3. This message would be written into a text file
    4. The AntlrProgram uses Rome grammar, ANTLR, and a visitor pattern in order to read in the stream of characters, tokenize each element of the text, assign value to each element of the text, and categorize each element of the text
      • input = FileStream(argv[1])
      • lexer = RomeLexer(input)
      • stream = CommonTokenStream(lexer)
      • parser = RomeParser(stream)
      • tree = parser.rome()
      • visitor = MercuryVisitor()
      • myMessages = visitor.getMessages(tree)
    5. myMessages is written into the RomeOutput text file
    6. The Flask web app then reads in each element and assigns it to its respective list
    7. Each list is then displayed via the web app
